  • Professional Photography Portfolio for Esko Photography in Las Vegas, Nevada

    Full Stack Website created with React/Redux/React-Router/MongoDB/Mongoose/Node

    Create with React, React-Router, Redux, MongoDB, Mongoose, Express


    Requirements for this site:

    • Site must provide the owner with a way to add work to his portfolio himself.
    • Work added must automatically be included to the portfolio page.
    • A booking application for site must be created that notifies the owner when a potential client schedules a booking.
    • Site must visually represent the companies image, with bright contrasts, and a landing page that grabs your attention, but isnt crowded.

    For this project I built a server, and created a private API with mongoDB, mongoose, and express, so I could call this API to populate the portfolio page with the photographers images, and provide the ability to add more images. The owner didnt want to have to sign into the site to add images, and didnt want anything on the site that suggested that this option even existed. The site also required the creation of a booking application so potential clients could when schedule appointments, and the notification be automatically sent to the owner by email.

    Collaborate

    The MERN stack application that utilizes third-party APIs. A group project done remotely, creating an mini social networking site for organzing group travel. The site allows users to collaborate together to plan out the activites for their group vacation, no matter the distance between them. The user can vote on the activities the other group members have contributed, and in the end the app creates the schedule for the group, with the activities, and destinations that were voted in by the whole group. The number of activites depends on the length of time the group will be staying somewhere, and the distance the group sets as a parameter for the search. Ultimately, allowing everyone to take part in the planning aspect of a group vacation.
